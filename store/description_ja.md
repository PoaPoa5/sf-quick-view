# SF Peek — Chrome Web Store 掲載文（日本語）

## 短い説明（132文字以内）
```
Salesforce管理者・開発者向けのクイック検査ツール。メタデータ閲覧、SOQLクエリ実行、ERダイアグラム表示、ファイル検索をブラウザだけで完結。
```

## 詳細説明

**SF Peek** は、Salesforce 管理者・開発者がブラウザから離れることなく、組織のメタデータ・自動化ロジック・ファイルに即座にアクセスできる Chrome 拡張機能です。

### 機能一覧

**オブジェクト・フィールド参照**
標準・カスタムを問わず任意のオブジェクトのフィールド情報（API 参照名・データ型・表示ラベル・必須フラグなど）を一覧表示。ワンクリックで CSV エクスポートが可能です。

**フィールドビューワー**
現在開いている Salesforce レコードのすべてのフィールド値を即座に確認できます。Setup に移動しなくてもデータの問題をその場でデバッグできます。

**SOQL ランナー**
ポップアップから直接 SOQL クエリを記述・実行。結果はスクロール可能なテーブルで表示され、クエリ履歴はローカルに保存されるためワンクリックで再実行できます。

**ER ダイアグラム**
オブジェクト間のリレーションシップをインタラクティブなダイアグラムで可視化。ノードをドラッグして配置を変更したり、マウスホイールでズーム・パンが可能です。主従関係とルックアップを色分けして表示。複雑なスキーマはフルページで表示できます。

**自動化ロジック確認**
任意のオブジェクトに紐づく Apex トリガー・入力規則・ワークフロールール・フローを、各 Setup ページに移動することなく一覧で確認できます。

**ファイル検索**
ファイル名で ContentDocument レコードを検索。個別ダウンロードまたはすべての結果を ZIP で一括ダウンロードできます。

### プライバシーについて
SF Peek は、現在ログイン中の Salesforce 組織とのみ通信します。外部サーバーへのデータ送信は一切ありません。SOQL 履歴はお使いのデバイスにのみ保存されます。

### 動作要件
- Salesforce 組織（エディション不問）
- Chrome ブラウザ（同一ウィンドウで Salesforce にログイン済みであること）
