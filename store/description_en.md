# SF Peek — Chrome Web Store Listing (English)

## Short Description (≤132 characters)
```
A quick inspection and admin tool for Salesforce. Browse metadata, run SOQL, visualize ER diagrams, and search files.
```

## Detailed Description

**SF Peek** gives Salesforce administrators and developers instant access to org metadata, automation logic, and files — all without leaving the browser.

### Features

**Objects & Fields**
Browse any standard or custom object and inspect every field: API name, data type, label, and whether it's required. Export the full field list as CSV in one click.

**Field Viewer**
Open any Salesforce record and instantly see all its field values. Useful for debugging data issues without navigating Setup.

**SOQL Runner**
Write and execute SOQL queries directly from the popup. Results are displayed in a scrollable table. Your query history is saved locally so you can re-run past queries with one click.

**ER Diagram**
Visualize object relationships as an interactive diagram. Drag nodes to rearrange, zoom and pan with the mouse wheel, and distinguish master-detail vs. lookup relationships at a glance. Open a full-page view for complex schemas.

**Background Logic**
See all Apex Triggers, Validation Rules, Workflow Rules, and Flows associated with any object — without navigating to each Setup page individually.

**File Search**
Search ContentDocument records by file name. Preview results and download individual files or batch-download all results as a ZIP archive.

### Privacy
SF Peek communicates only with the Salesforce org you are already logged in to. No data is sent to any external server. SOQL history is stored locally on your device only.

### Requirements
- A Salesforce org (any edition)
- Chrome browser, logged in to Salesforce in the same window
